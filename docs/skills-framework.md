# 框架技能

## Django

| 技能 | 作用 |
| --- | --- |
| django-patterns | Django 架构模式、REST API 设计（DRF）、ORM 查询优化和项目结构 |
| django-security | Django 安全最佳实践，认证、授权、CSRF、SQL 注入防护 |
| django-tdd | Django 测试策略，使用 pytest-django、TDD 方法论和 factory_boy |
| django-verification | Django 项目验证循环：迁移检查、测试运行、lint 和部署就绪验证 |

## Laravel

| 技能 | 作用 |
| --- | --- |
| laravel-patterns | Laravel 架构模式、路由/控制器/服务层设计和 Eloquent ORM 最佳实践 |
| laravel-security | Laravel 安全最佳实践，认证/授权、验证、XSS 和 SQL 注入防护 |
| laravel-tdd | Laravel 测试驱动开发，使用 PHPUnit 和 Pest 框架 |
| laravel-verification | Laravel 项目验证循环：环境检查、测试、代码风格和部署验证 |

## Spring Boot

| 技能 | 作用 |
| --- | --- |
| springboot-patterns | Spring Boot 架构模式、REST API 设计、依赖注入和分层架构 |
| springboot-security | Spring Security 最佳实践，认证/授权、JWT、OAuth2 和 CSRF 防护 |
| springboot-tdd | Spring Boot 测试驱动开发，使用 JUnit 5、Mockito 和 Testcontainers |
| springboot-verification | Spring Boot 项目验证循环：构建、类型检查、测试和部署就绪验证 |
