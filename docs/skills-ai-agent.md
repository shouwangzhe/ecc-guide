# AI 与 Agent 技能

## Claude API 与集成

| # | 技能 | 作用 |
| --- | --- | --- |
| 1 | claude-api | Anthropic Claude API 的 Python 和 TypeScript 集成模式，包括流式、工具调用和多轮对话 |

## 成本与性能优化

| # | 技能 | 作用 |
| --- | --- | --- |
| 1 | cost-aware-llm-pipeline | LLM API 使用的成本优化模式 — 模型路由、缓存、批处理和预算控制 |
| 2 | iterative-retrieval | 渐进式上下文检索模式，逐步精炼检索结果以减少 token 消耗 |

| 5 | strategic-compact | 在逻辑断点建议手动上下文压缩，优化长会话的上下文使用 |
